有两个按钮分别触发两个动画,当我按下按钮A，所触发的动画还未结束时，按钮A无法被再次触发，是否是主线程被阻塞了？

代码如下：

版本二没有类似的情况, 简单的重置计时器开始. 本应该同样的效果, 有什么错的地方？

// 版本二
-(IBAction)fadeUsingOld {
NSLog(@”V2: Clicked …”);
[myLabel setAlpha:1.0];
[UIView beginAnimations:nil context:nil];
[UIView setAnimationDuration:1.5];
[myLabel setAlpha:0.0];
[UIView commitAnimations];
}

最佳答案

animateWithDuration:animations: block 并不会阻塞主线程. 你遇到的这种情况, 是因为在默认情况下
在执行新的 animation block 时，交互（响应事件）是被禁止的，
解决方法，你可以重写 passing UIViewAnimationOptionAllowUserInteraction (calling animationWithDuration:delay:options:animations:completion)
